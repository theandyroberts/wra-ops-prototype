# WRA Ops Clickable Prototype

Open `index.html` in a browser.

This is a low-fidelity prototype for workflow review only. It has no backend, no real integrations, and no saved data.

## Review Path

1. Start on `Home`.
2. Open `Jobs`.
3. Click `Radical / Lexus`.
4. Review the job tabs:
   - Overview
   - Dates
   - Crew
   - Locations
   - POA
   - Paperwork
   - Airspace
   - Notes
   - Billing
   - Archive
5. Open `Calendar`.
6. Open `Resources`.

## What To Review

- Does the navigation match how producers think?
- Does the default Jobs view show the right information?
- Does the job workspace feel organized?
- Are Locations, POAs, Paperwork, Crew, and Calendar in the right places?
- What feels missing or too busy?
- What should be renamed?
